# Honeybee Café

A responsive, multi-page website for Honeybee Café, featuring a menu, dynamic order form, order confirmation, random bee/honey fact generator, and contact information.

## Features

- **Responsive Design:** Mobile, tablet, and desktop layouts using CSS flexbox and media queries.
- **Menu & Order Form:** Users can browse honey varieties and submit orders with dynamic item addition.
- **Order Confirmation:** Displays submitted order details and customer info.
- **Random Fact Generator:** Shows a new bee/honey fact on each click.
- **Navigation:** Fixed header with accessible navigation menu.
- **Contact & Location:** Footer includes café contact info and embedded Google Map.

## Tech Stack

- HTML5
- CSS3 (custom styles, flexbox, media queries)
- JavaScript (vanilla, DOM manipulation, sessionStorage)

## Folder Structure

```
css/                    # Stylesheets 
img/                    # Images and icons
js/                     # JavaScript
order.html              # Order form page
order-confirmation.html # Confirmation page
about-us.html           # About and fact generator
menu.html               # Menu page
index.html              # Home page
```

## Setup & Usage

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. No build steps or server required; all pages are static HTML.
